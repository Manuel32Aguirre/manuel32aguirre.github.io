Among Bugs

Juego interactivo de trivias con arquitectura full-stack, JWT y despliegue en Azure

Java Spring Boot Angular Docker Azure

Among Bugs es un juego de trivias interactivo que integra un backend robusto desarrollado con Spring Boot y un frontend dinámico en Angular. La aplicación presenta una mecánica de trampas que hace el juego más desafiante y entretenido. Está completamente desplegada en Microsoft Azure con autenticación JWT para usuarios seguros.

Características del Juego

Experiencia de juego interactiva y desafiante

Mecánica de Trampas

Sistema único de trampas que desafía al jugador y añade complejidad estratégica al juego de trivias tradicional.

Sistema de Puntuación

Cálculo dinámico de puntos basado en respuestas correctas, tiempo de respuesta y trampas evadidas.

Autenticación de Usuarios

Sistema de registro e inicio de sesión con JWT para mantener sesiones seguras de los jugadores.

Estadísticas y Leaderboard

Seguimiento de estadísticas del jugador y tabla de posiciones global para comparar desempeño.

Interfaz Responsiva

Frontend en Angular con diseño responsive que funciona perfectamente en desktop, tablet y dispositivos móviles.

Despliegue en Azure

Aplicación completamente desplegada en Microsoft Azure con alta disponibilidad y escalabilidad.

Arquitectura Técnica

Diseño full-stack moderno y escalable

1
Backend Spring Boot

APIs REST con arquitectura por capas, validación de datos y documentación Swagger

2
Seguridad JWT

Autenticación stateless con JWT, Spring Security y roles para control de acceso

3
Frontend Angular

Interfaz dinámica con TypeScript, componentes reutilizables y gestión de estado

4
Despliegue Azure

Containerizado con Docker, CI/CD con GitHub Actions y hosting en Azure

Stack Tecnológico Detallado

Backend: Java 17, Spring Boot 3.x, Spring Security, JPA/Hibernate
Autenticación: JWT, OAuth2 ready, Spring Security configuration
Frontend: Angular 16+, TypeScript, Bootstrap, componentes dinámicos
Base de Datos: MySQL/PostgreSQL con JPA para persistencia
Testing: JUnit 5, Mockito, MockMvc para pruebas completas
DevOps: Docker, Docker Compose, GitHub Actions, Azure deployment
