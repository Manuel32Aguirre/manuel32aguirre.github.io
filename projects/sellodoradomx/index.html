SelloDoradoMX

Plataforma web para el descubrimiento y apoyo de comercios locales

Spring Boot PostgreSQL Spring Security JWT Docker

Desarrollado durante el Hackathon Talent Land 2026, SelloDoradoMX es una solución integral para digitalizar la economía local. El backend robusto permite gestionar negocios, catálogos de productos y la interacción social entre usuarios y comercios.

Backend de Alta Disponibilidad

APIs REST optimizadas para la búsqueda geográfica y categorización de comercios locales.

Impacto y Desarrollo

  • Exposición de APIs REST para gestión de negocios, productos, favoritos y calificaciones.
  • Implementación de búsqueda avanzada por categoría y ubicación geográfica.
  • Sistema de autenticación y autorización seguro con Spring Security y tokens JWT.
  • Infraestructura contenerizada con Docker para asegurar despliegues consistentes.
  • Integración de canales de atención al usuario y validación de comercios.
