SelloDoradoMX

Plataforma web de comercios locales con APIs REST, autenticación JWT y sistema de favoritos y calificaciones

Java Spring Boot PostgreSQL Docker JWT

Plataforma web diseñada para el Hackathon Talent Land 2026 que permite a los usuarios descubrir, calificar y guardar sus comercios locales favoritos. El backend expone APIs REST para la gestión de negocios, productos, favoritos, calificaciones y búsqueda inteligente por categoría y ubicación geográfica.

Características Principales

Funcionalidades clave de la plataforma

Gestión de Negocios

APIs para crear, actualizar y listar negocios locales con información como categoría, ubicación, horarios y descripción.

Búsqueda Inteligente

Sistema de búsqueda por categoría y ubicación geográfica para que los usuarios encuentren comercios relevantes fácilmente.

Sistema de Favoritos

Los usuarios pueden guardar sus comercios favoritos para acceso rápido y personalizado.

Calificaciones y Reseñas

Sistema de calificación que permite a usuarios evaluar comercios y compartir experiencias con la comunidad.

Autenticación JWT

Autenticación segura con JWT tokens y Spring Security para proteger el acceso a los endpoints de la API.

Arquitectura Escalable

Diseño modular por capas siguiendo buenas prácticas de desarrollo para facilitar mantenimiento y escalabilidad.

Stack Tecnológico

Tecnologías utilizadas en el desarrollo

Backend

Java 17+ con Spring Boot framework, Spring Security para autenticación y validación de datos.

Base de Datos

PostgreSQL con JPA/Hibernate para mapeo objeto-relacional y consultas eficientes.

Containerización

Docker para empaquetar la aplicación y garantizar consistencia entre entornos.

Seguridad

JWT (JSON Web Tokens) para autenticación stateless y protección de APIs.

Características Técnicas

Validación de Datos: Anotaciones JSR 380 para validación robusta de entradas
Documentación: Swagger/OpenAPI para documentación interactiva de APIs
Manejo de Errores: Exception handling centralizado con respuestas consistentes
Paginación: Endpoints paginados para manejo eficiente de datos
Testing: Tests unitarios con JUnit 5 y Mockito
Deployment: Containerizado con Docker para despliegue simplificado
